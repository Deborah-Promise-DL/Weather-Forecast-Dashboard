# Weather Forecast Application

This project is a weather forecast application developed using HTML, Tailwind CSS, and JavaScript.

## Features
- Search weather by city name
- Current location weather using Geolocation
- 5-day weather forecast
- Weather condition icons
- Temperature unit toggle
- Extreme temperature alerts
- Input validation and error handling
- Responsive UI

## Technologies Used
- HTML
- Tailwind CSS
- JavaScript
- OpenWeatherMap API

## How to Run
1. Download or clone the repository
2. Open index.html in a browser
3. Enter a city name or allow location access
4. Use the unit toggle to switch temperature units
5. Click recent searches to quickly view weather again

Note: Replace the API key in `script.js` with your own OpenWeatherMap API key.
